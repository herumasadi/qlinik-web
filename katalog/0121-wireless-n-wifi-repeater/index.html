Solusi praktis untuk memperluas jangkauan sinyal WiFi di rumah atau kantor Anda. Alat ini menangkap sinyal dari router utama dan memancarkannya kembali ke area yang sebelumnya tidak terjangkau (Blank Spot).

Spesifikasi Teknis:

  • Standar Wireless: IEEE 802.11n/g/b
  • Kecepatan: Hingga 300 Mbps
  • Frekuensi: 2.4 GHz
  • Antena: Integrated Antennas
  • Port: 1x LAN Port (RJ45)

Keunggulan:

  • Desain Wall-Plug: Langsung dicolokkan ke stopkontak dinding, ringkas tanpa kabel berantakan.
  • Setup Mudah: Mendukung fitur WPS untuk sinkronisasi cepat dengan router utama.
  • Mode Fleksibel: Bisa digunakan sebagai Repeater (penguat sinyal) maupun Access Point (mengubah kabel LAN menjadi WiFi).
